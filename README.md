## Scripts disponíveis

No diretório do projeto você pode rodar: 

### `npm start`

Roda o app.<br />
Open [http://localhost:3000](http://localhost:3000) 

A página será recarregada a cada edição

### `npm test`

Roda test runner

### `npm run build`

Builda a aplicação, minificando css, js, html etc

### `npm run eject`
